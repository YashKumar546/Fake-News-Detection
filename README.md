## Project Overview

This project uses Natural Language Processing (NLP) and Machine Learning to classify news articles as **Fake** or **Real**. The text data is converted into numerical features using TF-IDF Vectorization, and a Logistic Regression model is trained for classification.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset

- Fake and Real News Dataset
- Source: Kaggle

## Features

- Load and combine datasets
- Preprocess news text
- TF-IDF vectorization
- Train Logistic Regression model
- Evaluate model performance
- Predict news labels
- Save trained model and vectorizer
